## 3. 処理フロー

### 3.1 タスク登録フロー

```
[ユーザー] タイトル・カテゴリ・優先度を入力して「登録する」を押す
        ↓
[Streamlit] タイトルが空でないかチェック（空の場合はエラー表示して停止）
        ↓
[Streamlit] POST /tasks にリクエスト送信
        ↓
[FastAPI] リクエストを受け取る
        ↓
[FastAPI] IDと登録日時を生成し、ステータスを「未完了」に設定する
        ↓
[FastAPI] tasks.json に追記保存
        ↓
[FastAPI] 保存した1件のデータを Streamlit に返す
        ↓
[Streamlit] 登録完了メッセージを表示する
```

### 3.2 一覧表示フロー

```
[ユーザー] サイドバーで「タスク一覧」を選択
        ↓
[Streamlit] GET /tasks にリクエスト送信
        ↓
[FastAPI] tasks.json を読み込む
        ↓
[FastAPI] 登録日時の降順でソートして返す
        ↓
[Streamlit] テーブル形式で一覧表示する
```

### 3.3 詳細表示フロー

```
[ユーザー] 一覧画面で「詳細を見る」ボタンを押す
        ↓
[Streamlit] 選択したタスクの ID を保持
        ↓
[Streamlit] GET /tasks/{id} にリクエスト送信
        ↓
[FastAPI] tasks.json から該当IDを検索して返す
        ↓
[Streamlit] 詳細画面に全項目を表示する
```

### 3.4 完了更新フロー

```
[ユーザー] 詳細画面で「完了にする」ボタンを押す
        ↓
[Streamlit] PATCH /tasks/{id}/complete にリクエスト送信
        ↓
[FastAPI] tasks.json から該当IDを検索する
        ↓
[FastAPI] ステータスを「完了」に更新してファイルを上書き保存する
        ↓
[FastAPI] 更新後のデータを Streamlit に返す
        ↓
[Streamlit] 詳細画面を再表示する（「完了にする」ボタンが非表示になる）
```

## 4. データ設計

### 4.1 JSONファイル仕様

| 項目 | 内容 |
|---|---|
| ファイル名 | tasks.json |
| 保存場所 | data/tasks.json |
| フォーマット | JSON配列 |
| 文字コード | UTF-8 |
| 初期状態 | ファイルが存在しない場合は空のリスト `[]` として扱う |

### 4.2 データ構造

```json
[
  {
    "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "created_at": "2026-05-11T10:00:00+09:00",
    "title": "議事録を作成する",
    "category": "仕事",
    "priority": "高",
    "status": "未完了"
  }
]
```

### 4.3 各フィールドの仕様

| フィールド名 | 型 | 生成タイミング | 生成元 |
|---|---|---|---|
| id | 文字列（UUID4） | 保存時にバックエンドで生成 | Python `uuid.uuid4()` |
| created_at | 文字列（ISO 8601） | 保存時にバックエンドで生成 | Python `datetime.now()` |
| title | 文字列 | ユーザー入力 | Streamlit フォーム |
| category | 文字列（仕事／プライベート／その他） | ユーザー入力 | Streamlit セレクトボックス |
| priority | 文字列（高／中／低） | ユーザー入力 | Streamlit セレクトボックス |
| status | 文字列（未完了／完了） | 登録時は「未完了」固定。完了更新時に「完了」に変更 | バックエンド |

---

## 5. エラー処理方針

| エラーの種類 | 発生箇所 | 対処方法 |
|---|---|---|
| タイトルが空 | Streamlit | エラーメッセージを表示し、送信しない |
| JSONファイルが存在しない | FastAPI | 空のリストとして処理を続行する |
| 指定IDが見つからない（詳細取得） | FastAPI | HTTPステータス 404 を返す |
| 指定IDが見つからない（完了更新） | FastAPI | HTTPステータス 404 を返す |
